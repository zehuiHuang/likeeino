## 功能特点

### 多智能体协作
- 采用专门化的代理分工合作完成复杂任务
- 支持最多20次迭代的Plan-Execute-Replan循环

### 文件操作能力
- 支持CSV文件的各种操作
- 可以读取、处理和生成新的电子表格文件
- 自动管理工作目录和临时文件

### 流式处理
- 支持实时输出处理结果
- 提供流畅的用户体验

### 可追踪性
- 集成Coze Loop回调进行执行跟踪
- 完整记录执行过程和状态变化

### 灵活配置
- 通过环境变量配置输入目录和工作目录
- 支持自定义任务参数

## 技术实现

### 上下文管理
程序使用 Context 传递任务相关信息：
- 文件路径
- 工作目录
- 任务ID
- 预览文件列表

### 错误处理
- 全面的错误检查和处理机制
- 详细的日志记录便于调试

## 总结

Excel Agent 展示了现代AI应用开发的强大能力，通过自然语言指令就能完成复杂的电子表格处理任务。其模块化设计和多智能体架构使得系统具有良好的扩展性和维护性，可以轻松适应更多类型的文件处理需求。
